# Dust

**[▶ Play in browser](https://nedajvoda01-dotcom.github.io/Dust/)** — open `index.html` served from the repo root, or run locally (see below).

**Source:** <https://github.com/nedajvoda01-dotcom/Dust>

---

A procedurally simulated alien world with no UI, no objectives, and no scripted events.
Walk, observe, endure.

## Run locally

```bash
pip install -r requirements.txt
python server.py          # starts on http://localhost:8765
```

Then open <http://localhost:8765> in a browser.
